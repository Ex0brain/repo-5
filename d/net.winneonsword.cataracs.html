Description

  • Cataracs is a realtively simple lockscreen clock / widget that features a minimal clock centered on the lockscreen. The theme is designed to not obscure the background wallpaper, but to show it off.

    iPhone 6/6 Plus users: If you intend to use the blur, you must change the approriate option in the options.js file, otherwise the blur may be off proportion. (version 1.3+)

    The name and request for Cataracs comes from the Reddit user /u/rstgyx. The original version of this theme is called Addict by @Nacknaw on deviantART.

    Designed for GroovyLock and LockHTML3/4.
