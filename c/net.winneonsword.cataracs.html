1.3

  • Added battery support, based on the battery mod by the reddit user /u/_guesswho. This is enabled by default in the options.js.

  • Added a horizontal clock. This is disabled by default in the options.js.

  • Added a temperature counter. This is disabled by default in the options.js. You can set your woeid (location) and your desired unit (C or F) in the options.js as well.

  • Removed the need to set an option for iPhone 6/6 Plus support, unless you intend to use the blur. If you use the blur and are not on a 4, 4S, 5, or 5S, you /must/ set the appropriate option in the option.js.

  • Added 4 new standalone options in the options.js. These include: border width, font family, x / y position, and a scale (size) multiplier.

1.2

  • Added several new options for you to mess around with!

  • Added a spacing option to control the spacing between the widget and the top of your screen. This is set to 160 pixels by default. You can change this option in the options.js.

  • Added a date. You can enable this date in the options.js. Part of the date code goes to /u/_guesswho! Make sure to give him a virtual high-five!

  • Added a blur that is disabled by default. You can enable this blur and change its blur radius in the options.js.

  • Added a shadow that is disabled by default. This shadow encompasses the widget. You can enable this shadow in the options.js.

  • Added a background colour option that is set to transparent by default. You can change the colour of this option in the options.js. For more colour information, go to http://www.colorpicker.com/ for hex codes.

  • Added a background colour opacity option in the options.js. This is set to 1.0 by default, and will only be used if the background colour is anything other than transparent.

  • Added a new dependency, groovyAPI. groovyAPI allows developers like myself to use abilities that normal JavaScript could not do with iOS. One of the things that I will be taking advantage in this API is notification support, finally!

  • Added a check to see if there is notifications on the lockscreen, and if so, moves the Cataracs widget up to the top of the screen.
